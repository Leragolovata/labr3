# 📘 Лабораторна робота №3

## 📌 Короткий опис задачі

Необхідно створити програму мовою C++, яка працює з квадратною матрицею розміру 4×4.

### Програма повинна:

- 🔹 ввести елементи матриці з клавіатури;
- 🔹 перевірити кожен елемент матриці;
- 🔹 визначити кількість додатніх, від’ємних та нульових елементів;
- 🔹 вивести результати підрахунку на екран.

---

## ⚙️ Опис алгоритму розв’язання

Алгоритм роботи програми складається з таких кроків:

1. Початок роботи програми.

2. Оголошення змінних:
   - матриця a[4][4];
   - змінні positive, negative, zero для підрахунку елементів.

3. Введення елементів матриці користувачем.

4. Перебір елементів матриці за допомогою вкладених циклів.

5. Перевірка кожного елемента:
   - якщо елемент більший за 0 — positive++;
   - якщо елемент менший за 0 — negative++;
   - якщо елемент дорівнює 0 — zero++.

6. Виведення результатів підрахунку.

7. Завершення роботи програми.

---

## 🧠 Пояснення основних етапів роботи програми

### 🔢 1. Введення матриці

Користувач вводить 16 чисел, які формують квадратну матрицю 4×4.

Для введення використовується вкладений цикл for, який проходить по всіх рядках і стовпцях матриці.

---

### 🔍 2. Перебір елементів матриці

Після введення даних програма проходить по всіх елементах матриці за допомогою вкладених циклів, щоб перевірити кожне число окремо.

---

### ➕➖ 3. Перевірка елементів

Для кожного елемента виконується перевірка:

if (a[i][j] > 0)
    positive++;
else if (a[i][j] < 0)
    negative++;
else
    zero++;
